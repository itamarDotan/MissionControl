
מנהל משימות (React + Vite)
אפליקציה פשוטה לניהול משימות: הוספה של משימות עם תיאור ותאריך יעד, עריכה, סימון כהושלם ומחיקה. הנתונים נשמרים ב־localStorage בדפדפן כך שישמרו אחרי רענון.

הרצת הפרויקט מקומית
התקנת תלותים:
npm install
הרצה במצב פיתוח (עם hot-reload):
npm run dev
בנייה לפרודקשן:
npm run build
להרצה מקומית של הבנייה (אופציונלי):
npm run preview
הערה: הפרויקט בנוי עם Vite ו־React. ודאו שדפדפןכם תומך ב־localStorage.

קומפוננטות ומחויבויותיהן
App.jsx: מנהל הסטייט הראשי — אחראי על רשימת המשימות, שמירה ב־localStorage ופיזור פונקציות הפעולה אל הילדים.
TodoInput.jsx: טופס הוספת משימה — קליטת תיאור ותאריך יעד, ולידציה לפני הוספה.
TodoList.jsx: תצוגת הרשימה — מקבל מערך משימות ומייצר TodoItem לכל אחת.
TodoItem.jsx: פריט משימה בודד — מציג טקסט, תאריך, תיבת סימון וכפתורי עריכה/מחיקה; מנהל מצב עריכה מקומי.
TodoFilters.jsx: כפתורי סינון — מחליפים בין תצוגות (הכל / פעיל / הושלם).
תכונות עיקריות
הוספת משימות עם תיאור ותאריך חובה.
עריכה של טקסט ותאריך (שמירה ידנית על ידי כפתור "שמור" או לחיצה על Enter).
סימון כהושלם ושמירת הסטטוס ב־localStorage.
סינון לפי סטטוס (הכל / פעיל / הושלם).
מגבלות ובאגים ידועים
הנתונים נשמרים רק ב־localStorage של הדפדפן; לא קיימת שכבה צד שרת או סנכרון בין מכשירים.
במצב מובייל הפריטים מוצגים בקונפיגורציית "מקופלת" — ייתכן שחוויית עריכה קטנה פחות נוחה במכשירים מסוימים.
אין אימות מתקדם מעבר לוולידציה בסיסית (למשל אין בדיקה לתאריכים בעבר).
מחיקת משימה היא סופית (אין סל מחזור). יש לגבות את ה־localStorage אם רוצים לשמור היסטוריה.
ייתכן שיהיו שינויים בתצוגה בין דפדפנים ישנים (העדיפו דפדפן עדכני).
קבצים חשובים: src/App.jsx, src/components/TodoInput.jsx, src/components/TodoList.jsx, src/components/TodoItem.jsx, src/components/TodoFilters.jsx.
